Test Case 1 of 11
ID      : C86452392
Title   : Administration - Enable/Disable - Disable Scan to Computer (WSD)
----------------------------------------------------------------------
Steps:

  Step 1:
    Disabled the Scan to Computer function
    - Access to EWS -> Security -> Printer Features.
    - Uncheck the Scan to Computer function.
    - Click on Apply button.
    - Check for the Scan to Computer app from control panel.
    Step applicable for Consumer devices with Scan to Computer.
    Expected Result:
      Verify the Scan to Computer has disappeared from the front panel.
      LED status, Sound and screen displays correctly.

  Step 2:
    Enabled the Scan to Computer function
    - Access to EWS -> Security -> Printer Features.
    - Checked the Scan to Computer function.
    - Click on Apply button.
    - Check for the Scan to Computer app from control panel.
    - Perform a Scan to Computer from control panle.
    Step applicable for Consumer devices with Scan to Computer.
    Expected Result:
      Scan to Computer app appears on the front panel.
      Scan to Computer job can be completed without error, and output is correctly.
      LED status, Sound and screen displays correctly.

  Step 3:
    Disabled the Web Scan function
    - Access to EWS -> Network -> Adcanced Settings > Remote Scan.
    - Turn off Network Initiated Scanning, eSCL, eSCL Secure and Web Scan.
    - Check Scan > Web Scan from EWS.
    Step applicable for Consumer devices with Web Scan.
    Expected Result:
      Verify the Web Scan has disabled from the front panel.
      LED status, Sound and screen displays correctly.

  Step 4:
    Enabled the Web Scan function
    - Access to EWS -> Network -> Adcanced Settings > Remote Scan.
    - Turn On Network Initiated Scanning, eSCL, eSCL Secure and Web Scan.
    - Check Scan > Web Scan from EWS.
    - Perform a Web Scan from EWS.
    Step applicable for Consumer devices with Web Scan.
    Expected Result:
      Web Scan is enabled on EWS page.
      Web Scan job can be completed without error, and output is correctly.
      LED status, Sound and screen displays correctly.

  Step 5:
    Disabled the WSD function
    - Access to EWS -> Network -> Adcanced Settings > Web Services (Microsoft).
    - Turn off WS-Scan.
    - Perform a WS-Scan job from PC (WSD).
    Step applicable for Consumer devices with WSD.
    Expected Result:
      Verify the WS-Scan has disabled and cannot perform from PC.
      LED status, Sound and screen displays correctly.

  Step 6:
    Enabled the WSD function
    - Access to EWS -> Network -> Adcanced Settings > Web Services (Microsoft).
    - Turn On WS-Scan.
    - Perform a WS-Scan job from PC (WSD).
    Step applicable for Consumer devices with WSD.
    Expected Result:
      WS-Scan is enabled on EWS page.
      WS-Scan job can be completed without error, and output is correctly.
